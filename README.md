# 📊 Pandas Examples Repository  

This repo contains **simple, beginner-friendly examples** of using the **pandas library** in Python 🐼.  

---

## ✨ Features
✔️ Create DataFrames  
✔️ Read CSV files  
✔️ Filter & Sort Data  
✔️ Group & Aggregate Data  
✔️ Merge & Concatenate Data  

---
🚀 Getting Started

Clone this repository
git clone https://github.com/your-username/pandas-examples.git
cd pandas-examples

Install dependencies
pip install pandas
